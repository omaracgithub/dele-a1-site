DELE A1 vs DELE A2 — 差异及2026年该选择哪个

更新于2026年6月 · 8分钟阅读

如果你刚开始学习西班牙语并想要认证你的水平，你可能会问是参加A1还是直接参加A2。这里我们为你解释这两个考试之间的实际差异，以便你做出正确的决定。

快速总结

CEFR级别入门基础

语法差异

DELE A1 — 仅现在时

  • 现在时的陈述语气（规则 + 常见不规则动词）
  • 有 / 在（存在和位置）
  • 喜欢（我喜欢 / 我不喜欢）
  • 冠词、物主代词、指示代词
  • 疑问词（什么、哪里、什么时候、怎么）

DELE A2 — 现在时 + 过去时 + 未来时

  • A1的所有内容，外加：
  • 现在完成时（我吃过，我去过）
  • 简单过去时（我吃了，我去了，我在）
  • 去 + 不定式（我要学习）
  • 正在 + 动名词（我在吃）
  • 比较级（比...更...，比...少...）
  • 基本命令式（来，看看，听）

词汇差异

A1：个人信息、数字、颜色、天/月、直系亲属、课堂物品、基本食物、房子（房间）。

A2：以上所有内容加上：城市和交通、购物（衣服、价格）、健康（医生、症状）、工作（职业、时间表）、休闲和空闲时间、天气。

考试差异

阅读理解

A1（45分钟，4个任务）：1-3行的文本：海报、表格、便条。

A2（60分钟，5个任务）：最多250个单词的文本：电子邮件、广告、简短文章。

听力理解

A1（20分钟，4个任务）：孤立的句子、数字、个人信息。非常慢。

A2（35分钟，5个任务）：短对话、信息、广告。速度适中。

书面表达

A1（25分钟，2个任务）：表格 + 30-40个单词的信息。

A2（50分钟，3个任务）：表格 + 60-70个单词的信息 + 70-80个单词的指导文本。

口头表达

A1（10分钟）：自我介绍，用简单句子描述照片，使用卡片进行引导对话。

A2（12分钟）：准备的独白，照片描述，模拟对话，自由对话。

申请西班牙国籍需要哪个？

DELE A2是最低要求。 A1不适用于居留国籍申请。如果你的目标是获得国籍，直接参加A2。

什么时候做A1有意义？

  • 刚开始：你学习西班牙语不到3个月，想要一个官方的第一个里程碑。
  • 需要动力：固定日期的考试迫使你定期学习。
  • 想熟悉格式：通过A1可以增强信心和经验，以便之后应对A2。
  • 学术要求：你的大学或交换项目接受A1。

什么时候直接跳到A2更合适？

  • 需要国籍： A1对你没有帮助。直接去A2。
  • 已经有基础：如果你学习了4-6个月或生活在西班牙语环境中。
  • 预算有限：如果你参加两个考试，费用是112欧元 + 130欧元。如果你只能负担一个考试，选择A2。
  • 时间有限：如果你已经有基础，经过2-3个月的密集学习可以直接去A2。

我的建议

如果你的目标是国籍，直接参加A2。如果你是从零开始并且不着急，A1是一个很好的第一步，为A2做准备。两个证书都是永久有效的，可以按任意顺序参加。
